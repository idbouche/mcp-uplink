# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-30

### Added
- Initial release of `@mcp-provider/cli`.
- `mcp-bridge` command for connecting local clients to remote MCP infrastructure.
- Secure environment variable forwarding (`X-Mcp-Env`).
- Support for `enabledTools` filtering.
- Comprehensive documentation and examples for Claude Desktop.
